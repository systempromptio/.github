### systemprompt.io

AI skills marketplace and governance infrastructure for Claude Code.

Browse, discover, and install curated AI skills, agents, and workflow templates at [systemprompt.io](https://systemprompt.io).

#### Open Source

| Project | Description |
|---------|-------------|
| [systemprompt-code-orchestrator](https://github.com/systempromptio/systemprompt-code-orchestrator) | MCP server for orchestrating AI coding agents (Claude Code CLI and Gemini CLI) |
| [systemprompt-mcp-server](https://github.com/systempromptio/systemprompt-mcp-server) | Production-ready MCP server with OAuth 2.1, task management, and integrations |
| [systemprompt-template](https://github.com/systempromptio/systemprompt-template) | Production AI agent mesh in 3 commands |
| [systemprompt-marketplace](https://github.com/systempromptio/systemprompt-marketplace) | Plugin marketplace for the systemprompt.io ecosystem |
